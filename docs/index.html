conda-forge documentation¶

What is conda-forge?¶

conda-forge is a community effort and a GitHub organization which contains repositories of conda recipes and thus provides conda packages for a wide range of software. The built distributions are uploaded to anaconda.org/conda-forge and can be installed with conda.

Missing a package that you would love to install with conda?

Chances are we have already packaged it for you. You can search for packages online. Look out for packages provided by our conda-forge organization.

Cannot find a package or only outdated versions of a package? - Everybody is welcome to contribute to our package stack!

  • To get started contributing packages, see Becoming involved.

  • To see our governance policies, see here.

  • If you find bugs, need help, or want to talk to the developers, use our mailing lists or chat rooms:

    • GitHub issues

    • Gitter channel

    • Mailing list
